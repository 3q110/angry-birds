# 🐦 愤怒的小鸟 - Angry Birds

一个使用原生 Canvas + JavaScript 开发的经典「愤怒的小鸟」网页小游戏，无需任何第三方依赖。

## 🎮 在线试玩

👉 **[点击这里在线试玩](https://3q110.github.io/angry-birds/)**

## 📖 游戏规则

### 目标
用弹弓发射小鸟，消灭关卡中所有的小猪即可过关。

### 操作方式

| 操作 | 说明 |
|------|------|
| **瞄准** | 点击/触摸小鸟并向后拖拽 |
| **发射** | 松开鼠标/手指，小鸟沿反方向飞出 |
| **轨迹预览** | 拖拽时会显示虚线预测弹道 |

### 游戏元素

- 🐦 **小鸟** — 你的弹药，每关数量有限，合理使用
- 🐷 **小猪** — 需要消灭的目标，全部消灭即过关
- 🪵 **木块** — 容易被摧毁（HP: 30）
- 🧊 **冰块** — 最脆弱的材料（HP: 20）
- 🪨 **石块** — 最坚固的材料（HP: 60）

### 计分规则

| 事件 | 得分 |
|------|------|
| 消灭一只小猪 | +100 分 |
| 摧毁一个方块 | +10 分 |

### 关卡

游戏共 **5 个关卡**，难度逐步递增：

1. **第 1 关** — 入门：1 只猪，木质结构
2. **第 2 关** — 进阶：2 只猪，冰木混合结构
3. **第 3 关** — 挑战：3 只猪，石木混合堡垒
4. **第 4 关** — 困难：4 只猪，双层复合结构
5. **第 5 关** — 终极：5 只猪，石头堡垒

### 功能按钮

| 按钮 | 说明 |
|------|------|
| 🔄 重新开始 | 重置当前关卡 |
| ⬅️ 上一关 | 返回上一关 |
| ➡️ 下一关 | 跳到下一关 |

## 🛠️ 技术栈

- **HTML5 Canvas** — 游戏渲染
- **原生 JavaScript (ES6 Module)** — 游戏逻辑
- **物理引擎** — 自研轻量级 2D 物理系统（重力、碰撞检测、弹性碰撞）

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/3q110/angry-birds.git

# 进入目录
cd angry-birds

# 启动本地服务器（任选一种）
python -m http.server 8080
# 或
npx serve .
```

然后在浏览器中打开对应地址即可。

## 📁 项目结构

```
angry-birds-minigame/
├── index.html          # 游戏入口页面
├── js/
│   ├── main-browser.js # 主程序入口（浏览器版）
│   ├── physics.js      # 物理引擎（重力、摩擦、弹跳）
│   ├── collision.js    # 碰撞检测系统
│   ├── bird.js         # 小鸟类
│   ├── pig.js          # 小猪类
│   ├── block.js        # 方块类（木/冰/石）
│   ├── slingshot.js    # 弹弓渲染与弹道预测
│   ├── scenes.js       # 背景场景（天空、云、山丘）
│   ├── ui.js           # UI 渲染（顶栏、胜利/失败界面）
│   ├── input.js        # 输入管理（鼠标 + 触摸）
│   └── level.js        # 关卡数据
└── start-server.bat    # Windows 一键启动脚本
```
